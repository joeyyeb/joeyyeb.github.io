## 🚀 How to Deploy on GitHub Pages (Step by Step)

### Step 1 — Create GitHub account
Go to [github.com](https://github.com) and sign up with your email. Use username **jfkay** to match your LinkedIn.

### Step 2 — Create a new repository
1. Click the **+** icon top right → **New repository**
2. Name it exactly: `jfkay.github.io`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload your files
1. Click **uploading an existing file** on the repo page
2. Drag and drop ALL files from this folder:
   - index.html
   - style.css
   - script.js
   - Joseph_Optimized_ProductManager.docx
   - Joseph_Optimized_DataScience.docx
3. Scroll down → click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repo → click **Settings**
2. Scroll down to **Pages** in the left sidebar
3. Under **Source** → select **Deploy from a branch**
4. Branch: **main** → folder: **/ (root)**
5. Click **Save**

### Step 5 — Your site is live!
Wait 2–3 minutes then visit:
👉 **https://jfkay.github.io**

That's your live URL. Share it everywhere — LinkedIn, Slack communities, job applications.

## ✏️ How to Update Content
Edit `index.html` directly on GitHub:
1. Go to your repo
2. Click `index.html`
3. Click the ✏️ pencil icon
4. Make changes → **Commit changes**
5. Site updates automatically in ~1 minute

## 🎨 How to Change Colors
Open `style.css` and edit the `:root` variables at the top:
```css
--gold:   #c9a84c;  /* Main accent color */
--cream:  #f0ead6;  /* Light text */
--bg:     #0c0c0e;  /* Background */
```
